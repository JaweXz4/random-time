# Random Time Generator

A Windows desktop application that generates random times between specified minimum and maximum values.

## Features

- 🎯 Generate random times between custom min/max ranges
- 🖥️ Modern Windows GUI built with WPF
- ⏱️ Support for hours, minutes, and seconds
- 📋 Copy generated time to clipboard
- 🎨 Clean and intuitive user interface

## Requirements

- Windows 7 or later
- .NET Framework 4.7.2 or higher

## Installation

1. Download the latest release
2. Run `RandomTimeGenerator.exe`
3. No installation required (portable application)

## Usage

1. Set your **Minimum Time** (hours, minutes, seconds)
2. Set your **Maximum Time** (hours, minutes, seconds)
3. Click **Generate Random Time**
4. The result will be displayed
5. Click **Copy to Clipboard** to copy the time

## Example

- Min: 1h 0m 0s
- Max: 3h 0m 0s
- Result: 2h 28m 49s
